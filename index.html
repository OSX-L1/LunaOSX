จัดการผู้ใช้

    ใบเสนอราคาที่บันทึกไว้ (สำหรับผู้ดูแลระบบ)

    แก้ไขข้อมูลผู้ใช้:

    สิทธิ์การมองเห็นหน้าเว็บ (Allowed Pages)

    เลือกหน้าที่ผู้ใช้คนนี้ได้รับอนุญาตให้เห็นและใช้งานได้ (จะ Override การตั้งค่าตามบทบาท)

    ราคาคำนวณเฉพาะ (Custom Prices)

    กำหนดราคาคำนวณเฉพาะสำหรับผู้ใช้คนนี้ในรูปแบบ JSON (หากกำหนดไว้จะใช้ราคานี้แทนราคา Global)
    **โครงสร้างราคา:**

    • **"pvc" / "wood"**: กำหนดราคาต่อหน่วยเป็นตัวเลขตรงๆ เช่น `"pvc": 355`
    • **"rail"**: กำหนดราคาตามประเภทรางและระบบ (wave_manual, m_cord, roman) เช่น `"rail": { "wave_manual": 230, "roman": 190 }`
    • **"roller"**: กำหนดราคาตามประเภทหลัก (Category) และรุ่นสินค้า (Type) เช่น `"roller": { "Sunscreen": { "LUMI 5%": { "price": 290 } } }`
